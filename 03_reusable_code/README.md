# Markup languages and reproducible programming in statistics

Deliverable 3: Reusable `R` code wrapped in an `R` package or `Shiny` app.

This package relates to statistics from the game of korfball, which I can imagine the graders to be unfamiliar with. For a very brief introduction to the game of korfball, see [this](https://www.croydonkorfball.com/rules.pdf) explanation from Croydon Korfball Club or [this](https://www.youtube.com/watch?v=GUtJB5jBOis) (slightly outdated) introductory video.
